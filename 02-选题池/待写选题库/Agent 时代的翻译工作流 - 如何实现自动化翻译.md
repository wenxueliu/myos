# Agent 时代的翻译工作流 - 如何实现自动化翻译

## 选题背景

Agent 时代的翻译，不再是"复制 - 粘贴 - 调 prompt"，而是自动化工作流。

> 好的翻译 Agent，应该让你忘记翻译过程的存在

## 核心问题

### 1. 工作流设计
- 文件自动检测与读取
- 内容分段与上下文管理
- 术语一致性处理
- 格式保留与还原

### 2. 质量控制
- 自动校验与纠错
- 人工审核节点设计
- 版本管理与回溯

### 3. 工具集成
- 与文档工具集成
- 与发布平台对接
- 多语言协作支持

## 核心观点

> Agent 的价值不是替代翻译，是消除翻译的摩擦

> 自动化工作流的核心是可靠，不是炫技
