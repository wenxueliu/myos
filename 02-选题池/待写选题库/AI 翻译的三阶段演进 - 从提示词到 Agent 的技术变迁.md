# AI 翻译的三阶段演进 - 从提示词到 Agent 的技术变迁

## 选题背景

宝玉提出 AI 翻译的三阶段框架：

> 提示词时代 → 推理模型时代 → Agent 时代

这个框架不仅适用于翻译，也适用于其他 AI 应用场景。

## 核心问题

### 1. 提示词时代
- 精心设计的 prompt 是关键
- 用户需要学习"提示词工程"
- 结果不稳定，依赖技巧

### 2. 推理模型时代
- 模型本身能力提升
- prompt 简化，更自然
- 推理能力增强，理解更准确

### 3. Agent 时代
- 自动化工作流
- 多步骤任务自主完成
- 人工干预最小化

## 核心观点

> 技术演进的 direction：让用户少操心，让 AI 多做事

> 每个阶段的跨越，都是能力门槛的降低

## 文章结构

1. **阶段一** - 提示词时代的特点与局限
2. **阶段二** - 推理模型带来的改变
3. **阶段三** - Agent 如何重新定义工作流
4. **对比** - 三个阶段的体验差异
5. **展望** - 下一个阶段可能是什么

## 待验证要点

1. 原文完整内容获取
2. 各阶段代表性工具/技术
3. 实际翻译质量对比
