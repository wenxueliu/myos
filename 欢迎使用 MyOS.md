# 欢迎使用 MyOS

这是我的个人操作系统 - 一个基于 Obsidian 的知识管理和生活操作系统。

## 📁 建议的文件夹结构

```
MyOS/
├── 00-Inbox/          # 临时收集区
├── 10-Daily/          # 每日笔记
├── 20-Projects/       # 项目笔记
├── 30-Areas/          # 责任领域
├── 40-Resources/      # 资源库
├── 50-Archives/       # 归档
└── Templates/         # 模板
```

## 🚀 快速开始

1. 创建你的第一篇每日笔记
2. 设置模板文件夹
3. 开始记录你的想法

## 💡 提示

- 使用 `Ctrl/Cmd + O` 快速切换笔记
- 使用 `[[ ]]` 创建双向链接
- 使用 `#标签` 分类笔记

*开始构建你的第二大脑吧！* 🧠
