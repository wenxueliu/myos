# OpenClaw: The Complete Guide to Building, Training, and Living with Your Personal AI Agent

> 作者：Claire Vo
> 发布日期：2026-03-31
> 来源：Lenny's Newsletter

## 核心观点

Claire Vo 构建了一支由 **9 个 AI Agent** 组成的团队来运行她的工作和生活。本文是完整的构建指南。

**副标题**：我构建了 9 个 AI Agent 来运行我的工作和生活。你也可以。

## OpenClaw 是什么

OpenClaw 是一个开源的个人 AI 助手，运行在本地机器上，可通过 Telegram 或 WhatsApp 等消息平台控制。

核心架构：
- 本地网关系统
- 独立 Agent，各有身份、工具和工作空间
- Agent 按 cron 任务计划工作
- 可以安装 skills、API 和 CLI 来与系统交互

## 部署选项

| 方案 | 特点 |
|------|------|
| **StartClaw/MyClaw/SimpleClaw** | 托管版本 |
| **Railway/DigitalOcean/Google Cloud** | VPS 部署 |
| **专用 Mac Mini 或笔记本** | 推荐（完全控制） |

| Agent | 角色 | 职责 |

| **Polly** | 个人助理 | 邮件、日历、Linear |
| **Finn** | 家庭管家 | 日程、协调 |
| **Max** | 营销 | 社交媒体、内容 |
| **Sam** | 销售 | CRM、线索筛选 |
| **Holly** | 客服 | 支持工单 |
| **Sage** | 课程运营 | 课程管理 |
| **Howie** | 制作人 | 播客制作 |
| **Kelly** | 开发者 | 编码任务 |
| **Q** | 教授 | 孩子教育 |

## 6 个入门工作流

1. **周末家庭协调** — Finn 主导
2. **热点研究 + 梗图生成** — Max
3. **PLG 注册分析与跟进** — Sam
4. **会议准备简报** — Polly
5. **支持文档生成** — Holly
6. **项目管理** — Polly/Kelly

## 安全建议

- **不要**在常用电脑上安装
- 初始使用只读 token
- 定期更新和安全审计
- 警惕提示词注入风险

## 成本
